# SETR-Robot-Telemetry

Este repositorio contiene la configuración y las instrucciones necesarias para la instalación y uso básico de Node-RED en Ubuntu. Se describe el proceso de instalación de Node.js, npm y Node-RED, así como el arranque del servidor y el acceso a la interfaz web tanto en local como desde otros equipos de la misma red. El proyecto está orientado al desarrollo de flujos visuales para la integración de comunicaciones (por ejemplo, MQTT) y la creación de interfaces gráficas mediante el Dashboard de Node-RED.
